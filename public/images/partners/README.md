# Logos des Partenaires

Ce dossier contient les logos des partenaires du CCEABT.

## Comment ajouter un logo

1. Placez le fichier image dans ce dossier (`public/images/partners/`)
2. Nommez le fichier selon l'une des conventions suivantes :

### Noms recommandés (par ordre de priorité)

#### Conseil d'administration :
- `padi.png` (ou .jpg, .jpeg, .webp, .svg)
- `chaine-de-lespoir.png`
- `fiadi.png`
- `odiae.png`
- `adesco.png`
- `ajt.png`
- `cdd.png`

#### Partenaires Techniques et Financiers (PTF) :
- `aesen.png`
- `afd.png`
- `ue.png` ou `union-europeenne.png`
- `pseau.png`
- `coalition-eau.png`
- `swa.png`
- `aafea.png`
- `endwaterpoverty.png`
- `ambassade-france-togo.png`
- `genda-water-alliance.png`
- `plan-international-togo.png`
- `seves.png`
- `cawst.png`

## Formats acceptés

- PNG (recommandé)
- JPG/JPEG
- WEBP
- SVG

## Gestion automatique des variations

Le système recherche automatiquement les images avec différentes variations de noms :
- Avec ou sans tirets
- Avec ou sans majuscules
- Avec différentes extensions
- Dans plusieurs emplacements

Si le logo n'est pas trouvé, le nom du partenaire sera affiché avec une initiale stylisée.

## Conseils

- Utilisez des images de haute qualité (minimum 200x200px)
- Préférez les logos sur fond transparent (PNG)
- Assurez-vous que le logo est bien centré dans l'image
- Les logos seront automatiquement redimensionnés pour s'adapter
