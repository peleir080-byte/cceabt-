# Guide d'Utilisation - Administration CCEABT

Ce guide détaille comment utiliser le panneau d'administration pour gérer le contenu du site web du CCEABT.

## 1. Accès au Panneau d'Administration

*   **URL** : `/cceabtadmin` (ou via le lien "Admin" discret en bas du site).
*   **Mot de passe par défaut** : `admin123`
*   **Connexion** : Entrez le mot de passe et cliquez sur "Se connecter".

---

## 2. Tableau de Bord Général

Une fois connecté, vous avez accès à 5 onglets principaux situés en haut :
1.  **Actualités** : Gestion des articles de blog et news.
2.  **Partenaires** : Gestion des logos et infos des partenaires.
3.  **Galerie** : Gestion des photos de la galerie média.
4.  **Ressources** : Gestion des documents téléchargeables.
5.  **Paramètres** : Modification de vos identifiants.

---

## 3. Gestion des Actualités (Onglet "Actualités")

Cet onglet permet d'ajouter des nouvelles qui s'afficheront sur la page "Actualités".

### Ajouter une actualité
1.  Remplissez le formulaire à gauche :
    *   **Titre** : Le titre de l'article.
    *   **Catégorie** : Choisissez le type (Plaidoyer, Eau, Formation...).
    *   **Date** : Ex: `15 Mars 2025`.
    *   **Image URL** : Lien vers une image (ex: `https://...`).
    *   **Extrait** : Un court résumé qui apparaît dans la liste.
2.  Cliquez sur le bouton **"Ajouter"**.

### Supprimer une actualité
*   Dans la liste de droite, cliquez sur l'icône **Corbeille** rouge à côté de l'article à supprimer.

---

## 4. Gestion des Ressources (Onglet "Ressources")

Cet onglet gère les documents affichés sur la page "Centre de Documentation".

### Ajouter un document
1.  Remplissez le formulaire :
    *   **Titre** : Nom du document.
    *   **Type** : PDF, Rapport, Guide...
    *   **Thème** : Ex: Hygiène, Eau potable.
    *   **Année** : Année de publication.
    *   **Lien Téléchargement** :
        *   Vous pouvez mettre un lien Google Drive, Dropbox, ou un lien direct vers un PDF hébergé.
2.  Cliquez sur **"Ajouter"**. Le document apparaîtra immédiatement sur la page publique.

---

## 5. Gestion des Partenaires (Onglet "Partenaires")

Gérez les logos et descriptions sur la page "Nos Partenaires".

### Ajouter un partenaire
1.  Remplissez :
    *   **Nom** : Nom de l'organisation.
    *   **Type** : Important pour le filtre (ONG Internationale, Nationale, Technique...).
    *   **Description** : Courte présentation.
    *   **Site Web** : Lien vers leur site.
2.  Cliquez sur **"Ajouter"**.

---

## 6. Gestion de la Galerie (Onglet "Galerie")

Cet onglet est spécifique pour la galerie photos (page "Media" ou "About").

*   **Ajouter une photo** :
    *   Donnez un **Titre**.
    *   **Upload Image** : Cliquez pour choisir une image depuis votre ordinateur.
    *   Cliquez sur **"Ajouter"**.
*   **Sauvegarder** : N'oubliez pas de cliquer sur le bouton bleu **"Sauvegarder"** en haut de cet onglet pour valider les changements de la galerie.

---

## 7. Paramètres & Sécurité

L'onglet **Paramètres** vous permet de changer vos identifiants.

*   **Modifier les identifiants** : Cliquez sur "Modifier", changez l'email ou le mot de passe, puis "Enregistrer".
*   **Note** : Ces informations sont stockées localement dans votre navigateur pour cette version de démonstration.

---

## Besoin d'aide ?
Si vous rencontrez un problème technique, essayez de rafraîchir la page (F5). Les données sont sauvegardées automatiquement.
